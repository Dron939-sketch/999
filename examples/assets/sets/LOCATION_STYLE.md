# Стиль локаций «под Фримена»

Локация у Фримена — **не фотореализм**, а графичный рисунок тушью. Персонаж
(сплошной чёрный) читается на светлом поле, вокруг — минимум жирных чёрных
форм. Наша ошибка была в градиентах и мягком свете — они дают «фото».

## Правила (обязательные)

1. **Никаких градиентов.** Только плоские заливки (`fill="#..."`, один тон на
   плоскость). Ни `radialGradient`, ни `linearGradient` для стен/пола/света.
2. **Светлое поле, чёрный субъект.** Фон — светло-серая «бумага» (#d4d7cf).
   Главные акценты — **жирная чёрная тушь** (#141410), толщина 4–8px.
3. **Тон — блоками, не переходом.** Глубину даём 3–4 плоскими тонами
   (потолок темнее, стены светлее, пол средний), разделёнными чёрным контуром.
4. **Тени — сплошные чёрные пятна** (флэт, ~0.8 α), а не мягкие размытия.
5. **Живая линия.** Всё, что нарисовано тушью, — через фильтр `#ink`
   (feTurbulence+displace), чтобы линия «дышала», как рисованная от руки.
6. **Высокий контраст, мало деталей.** Лучше 5 сильных чёрных форм, чем 50
   мелких. Дьявол в мелочах — но мелочи это акценты (надпись, зарубки), а не
   текстурная каша.
7. **Свет — если нужен — один флэт-полигон** полупрозрачного белого, жёсткими
   гранями (луч из окна), не радиальное свечение.
8. Зерно/виньетка/контраст накладываются рендером (config: `film-grain`,
   `vignette`, `mono-contrast`) — сам SVG держим чистым и графичным.

## Палитра

| Роль | Цвет |
|------|------|
| Бумага-фон | `#d4d7cf` |
| Светлая плоскость (задняя стена) | `#c9ccc3` |
| Средняя плоскость (боковые стены) | `#a2a59d` |
| Тёмная плоскость (потолок) | `#5f625b` |
| Пол | `#b0b3ab` |
| Тушь / контур | `#141410` |
| Сплошная тень | `#17170f` @ 0.8 |
| Луч света | `#f2f4ec` @ 0.35, флэт |

Референс: кадры Мистера Фримена — плоский светлый фон, жирный чёрный контур,
экстремальный контраст, графика вместо рендера.
