# Мистер Фримен — автоматизированный пайплайн (по коду)

Реверс-инжиниринг того, как делали **Mr. Freeman** (Flash/Animate, студийная
команда), и как каждый шаг воспроизведён **автоматически** в движке `animdsl`,
чтобы на выходе было то же качество без ручной покадровки.

| Шаг оригинала (Flash) | Что это | Автоматизация в animdsl |
|---|---|---|
| **1. Векторная рисованная графика** — жирная кистевая линия, плоские чёрные силуэты, белая маска-лицо | Художник рисует части персонажа вектором | SVG-части в `assets/characters/freeman_rig/` (маска, глаза, рот, тело, тонкие руки/ноги); плоский чёрный `#0e0e0e` + жирный контур маски |
| **2. Cutout-риг + символы** — тело собрано из переиспользуемых частей, крупная motion через трансформации | Аниматор двигает части, а не перерисовывает | Иерархический скелет `rig.json` (FK: локти/колени), позы интерполируются движком |
| **3. Покадровая мимика** — лицо и кисти рисуют по кадрам | Живость лица | Кости `eye_left/right`, `mouth` + библиотека выражений (`talk, wide, stern, shrug, point…`) |
| **4. Boil (живая линия)** — контур перерисовывают каждые ~2–3 кадра, линия «кипит» | Ручная дрожь линии | **Процедурно**: `feTurbulence`+`feDisplacementMap` на каждой части, а движок на каждом кадре меняет `seed` турбулентности (`apply_boil` в `renderer`) → контур дрожит сам |
| **5. Ограниченная анимация** — держим позы, плавные ease, тайминг под голос | Ритм монтажа | Позы + `wait` + `transition` + ease в таймлайне; смена планов `camera medium/close/extreme-close` |
| **6. Пост: плёнка** — зерно, виньетка, царапины, состаренный тон | Финальный «киношный» слой | `config { film-grain, vignette }` → `apply_film` в движке (детерминированное зерно по кадру + мягкая виньетка); царапины — в сет-SVG |

## Как собрать кадр «как у Фримена»

```
config {
    monochrome: true          // (по желанию) свести к 2 тонам
    film-grain: 0.34          // плёночное зерно
    vignette: 0.5             // затемнение краёв
}
```

Персонаж (`freeman_rig`) уже несёт «живую линию» (boil) — отдельно включать не
нужно, она активируется автоматически, т.к. в частях есть турбулентный фильтр.

## Что осталось для «один-в-один»

- **Кистевая толщина линии (толсто-тонко).** Сейчас контур ровной толщины;
  настоящая кисть даёт нажим. Приближение: переменная ширина обводки по частям.
- **Дуги и squash/stretch в motion.** Cutout-риг двигает части жёстко; живость
  добавит вторичная моторика и лёгкий squash (доработка движка).
- **Выразительные кисти** (растопыренные пальцы) как отдельные позы кисти.

## Файлы

- `assets/characters/freeman_rig/` — персонаж (части + `rig.json`).
- `assets/sets/freeman-stage.svg` — светлый фон-сцена (виньетка, царапины).
- `freeman-monologue.anim` — демо-монолог со всеми слоями пайплайна.
- Движок: `src/renderer/mod.rs` (`apply_boil`), `src/main.rs` (`apply_film`,
  `apply_monochrome`), `src/scene/mod.rs` (конфиг `film-grain`, `vignette`,
  `mono-contrast`).
