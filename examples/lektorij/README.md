# Лекторий → ролики Фреди (стиль Мистера Фримена)

Адаптация движка `animdsl` под видео к лекциям Лектория. К каждой лекции
делается короткий **чёрно-белый рисованный ролик**, где персонаж **Фреди**
говорит в камеру — в манере Мистера Фримена: резкий контраст, крупные планы,
минимум фона.

Видео **немое**. Закадровый голос берётся из серверного TTS лекции (репозиторий
`Dron939-sketch/Frederick`, `backend/blog_tts_routes.py`) и подмешивается на
этапе сборки. Это разделение — движок рисует, TTS озвучивает — и есть суть
адаптации.

## Конвейер

```
лекция (blog/lekciya-*.html)          TTS лекции (Frederick, .mp3)
        │                                     │
        ▼                                     │
tools/gen_lecture_anim.py                     │
        │  .anim (немой сценарий Фреди)        │
        ▼                                     │
animdsl render → result.mp4 (ч/б, без звука)  │
        │                                     │
        └──────────► tools/compose_video.sh ◄─┘
                            │
                            ▼
                   финальный ролик (видео + голос + титр)
```

## Персонаж Фреди

`examples/assets/characters/procedural/fredi.json` — лысый, круглые очки,
чёрная бабочка, чёрный смокинг. Ровно тот образ, что на обложке: интеллигентный,
собранный, чуть ироничный. Рендерится процедурно — ни спрайтов, ни SVG.

Под него в движок добавлены:

- `hair-style: Bald` — чистый блестящий череп (блик на темени);
- аксессуар `BowTie` — бабочка на воротнике;
- круглые линзы очков (аксессуар `Glasses`);
- `config { monochrome: true }` — постобработка кадра в контрастное ч/б («тушь»);
- направление `facing front` — Фреди смотрит прямо в камеру.

Трость с волчьей головой (есть на обложке) движок пока не умеет — это отдельная
задача (новый аксессуар `Cane` в `src/procedural`).

## Библиотека поз

Канон — в `fredi-lecture-template.anim` и в генераторе (`POSE_LIBRARY`). Все позы
фронтальные, лицом в камеру:

| Поза            | Смысл                                   |
|-----------------|-----------------------------------------|
| `fredi-calm`    | база, спокойное наблюдение              |
| `fredi-blink`   | моргание, микропауза                    |
| `fredi-think`   | задумчивость, рука к подбородку         |
| `fredi-explain` | открытый жест, объяснение               |
| `fredi-point`   | указание, «это про тебя»                |
| `fredi-stern`   | строгий акцент                          |
| `fredi-smirk`   | усмешка, «я знаю, к чему веду»          |
| `fredi-lean`    | наклон вперёд, усиление                 |

Меняете позы — правьте оба места синхронно (шаблон и генератор).

## Как сделать ролик к лекции

1. **Сгенерировать сценарий** из HTML лекции:

   ```
   python3 tools/gen_lecture_anim.py \
       ../dron939-sketch.github.io/blog/lekciya-2-frejd-psihodinamika.html
   ```

   Появится `examples/lektorij/lekciya-2-frejd-psihodinamika.anim`
   (заголовок → интро, каждый раздел «N. …» → доля, «Итоги» → финал).

2. **Отредактировать** реплики `VO:` в комментариях и подогнать паузы `wait`
   под длительность озвучки. Стиль — по методичке лекций
   (`blog/lektorij/METODICHKA.md` на сайте): цель — понимание, не пересказ.

3. **Проверить и отрендерить**:

   ```
   cargo run --release -- check  examples/lektorij/lekciya-2-frejd-psihodinamika.anim
   cargo run --release -- render examples/lektorij/lekciya-2-frejd-psihodinamika.anim -o fredi.mp4
   ```

4. **Собрать со звуком** (и титром):

   ```
   tools/compose_video.sh fredi.mp4 lekciya-2.mp3 lekciya-2-final.mp4 "Лекция 2. Фрейд"
   ```

## Пилот

`lekciya-2-frejd-psihodinamika.anim` — эталонный, вручную отшлифованный ролик
(лекция «Фрейд: психодинамическое направление» курса «Теории личности»).
Именно он рендерится по умолчанию в `Dockerfile` и в CI (`.github/workflows/render.yml`).

## Массовая генерация

Лекций ~720 в ~72 курсах. Прогнать генератор по всем:

```
for f in ../dron939-sketch.github.io/blog/lekciya-*.html; do
    python3 tools/gen_lecture_anim.py "$f"
done
```

Затем рендерить нужные и собирать со звуком. Тайминг у автогенерации
эвристический — под финальный монтаж паузы правятся под реальную озвучку.
