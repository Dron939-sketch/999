## Липсинк

Когда mp3 готов:
```
python3 tools/lipsync.py intro.mp3 --entity freeman -o lipsync.anim.inc
```
— скрипт снимет огибающую громкости и выдаст точную дорожку ртов
(`pose "talk"/"gab"/"idle"`), которой можно заменить блоки `speaks for` в
`.anim` (вставить содержимое файла на место реплик). Требуется ffmpeg/ffprobe.

## Сборка

```
cargo run --release -- render examples/lektorij/pereproshivka-intro.anim -o intro-video.mp4
tools/compose_video.sh intro-video.mp4 intro-voice.mp3 intro-final.mp4
```
SFX подмешиваются в любом редакторе (или вторым ffmpeg-проходом).
